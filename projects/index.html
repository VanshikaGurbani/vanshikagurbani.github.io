01

BioNER: Multi-task Biomedical NER

View Source

Built a multi-task learning model using BioBERT v1.1 to extract biomedical entities (genes, diseases, chemicals) from 5 benchmark corpora (BC2GM, BC4CHEMD, BC5CDR, JNLPBA, NCBI Disease). Compared single-task vs multi-task approaches, achieving F1=0.98 on BC2GM gene recognition and F1=0.99 on NCBI Disease extraction. Engineered a preprocessing pipeline with IOBES tagging, token alignment, and dynamic batching to support low-resource learning with strong cross-dataset generalization.

02

True Image Caption Generator

View Source

Built a hybrid image captioning system using InceptionV3 for visual feature extraction and a dual-mode Transformer supporting both greedy and beam search decoding. Trained on MS-COCO, Flickr8k, and Flickr30k datasets (120K+ labeled images), achieving a BLEU score of 0.80. Deployed as a Streamlit web app for real-time caption prediction from uploaded images.

03

Growth Mindset Causal Analysis

View Source

Estimated the Average Treatment Effect of growth mindset interventions on student achievement using 10K+ observational education records. Applied multiple causal estimators including Causal Forests, Inverse Propensity Weighting (IPW), and meta-learners (S-Learner, T-Learner, X-Learner) to ensure robust, estimator-agnostic conclusions. Results showed a consistent positive causal effect across all methods.
